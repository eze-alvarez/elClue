# El Clue
Anotador creado para facilitar la identificación del personaje, lugar y objeto.

Cada lista cuenta con items que tienen: 1 input checkbox, nombre, 1 input texto y 3 input color.

Se dispuso de un input texto el cual permite 10 caracteres de limite para observaciones, 3 input color para que el jugador en caso de ser necesario pueda elegir algun color para asociar a un posible jugador de la mesa.

Cuenta con un seleccionador de cartas las cuales pueden filtrarse por todas, descartadas y posibles.

Se adicionó un boton en la parte inferior del viewport para poder reiniciar todos los valores de las listas.
